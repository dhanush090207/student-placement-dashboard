# Student Placement Dashboard (Power BI)

This project is an interactive Student Placement Dashboard built using Power BI to analyze placement performance and trends.

## Features

* Total Students, Placed, and Unplaced overview
* Placement percentage KPI
* Department-wise placement analysis
* Company-wise hiring distribution
* Skill-wise placement insights
* CGPA vs Package analysis
* Interactive filters (Department, Gender, Status)

## Tools Used

* Power BI
* Data Modeling
* DAX (Data Analysis Expressions)
* Data Visualization

## Key Insights

* Placement Rate: 48.67%
* Highest Package: 7.5 LPA
* Top Skills: Excel, SQL, Java
* Companies Hiring: TCS, Infosys, Capgemini, Cognizant

## How to Use

1. Download the `.pbix` file
2. Open using Power BI Desktop
3. Interact with filters and visuals
